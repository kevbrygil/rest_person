# App Rest Person
Development of an REST API for CRUD operations of Persons

# Main Tecnologies used
- NodeJs Language
- Express Framework
- MongoDB Database
- The ODM Mongoose for mapper the objects and assign in a MongoDB document.

# Requirements
- DMongoDB Database
- Robo 3T(Optional)
- Nodejs with NPM
